# 리눅스 명령어 실습
- 현재 경로 보기 </br>
  ``` pwd ```
- 디렉토리 생성 </br>
  ``` mkdir mv_test ```
- 해당 디렉토리 이동 </br>
  ``` cd mv_test ```
- 파일, 디렉토리로 이동 </br>
  ``` mv hello.c mv_test ```
- 비어있는 파일 생성 </br>
  ``` touch test1.txt ```
- 파일 내용 중 처음/마지막 10줄 출력 </br>
  - ``` head hello.c ```
  - ``` tail hello.c ```
